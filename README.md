### WOORIN STOCK SYSTEM TRADER

## 개요

    한국투자증권 계좌를 대상으로 국내/해외 주식의 현황을 한 눈에 보기 쉽도록 하는것이 목적.

    - 실시간 매수는 기존 MTS에서 진행함.

    - 실시간 매도의 경우는 자동으로 선택 할 수 있도록 함.

    - 한국투자증권의 자산 탭이 명확하게 현재의 시점으로 보이지 않으므로 실제 자산을 볼 수 있도록 함.
